# 📘 서포트 벡터 머신 (SVM)

SVM은 선형/비선형 분류 및 회귀, 이상치 탐색에 모두 사용 가능한 강력한 모델입니다.

## 5.1 선형 SVM 분류

### 📌 핵심 개념
* **라지 마진 분류(Large Margin Classification)**: 클래스 사이의 폭이 가장 넓은 도로를 찾는 방식입니다.
* **서포트 벡터(Support Vector)**: 결정 경계(도로)의 가장자리에 위치하여 도로의 경계를 지탱하는 샘플들입니다.

### 📌 소프트 마진 분류 (Soft Margin Classification)
* **하드 마진**: 모든 샘플이 도로 밖에 있어야 함. 이상치에 매우 민감합니다.
* **소프트 마진**: 도로의 폭을 넓게 유지하는 것과 **마진 오류(Margin Violation)** 사이의 균형을 잡습니다.
* **규제 하이퍼파라미터 $C$**:
    * **$C$ 증가**: 마진 오류를 엄격하게 제한 → 도로 폭이 좁아짐 (과대적합 위험).
    * **$C$ 감소**: 마진 오류를 더 많이 허용 → 도로 폭이 넓어짐 (일반화 성능 향상, 과소적합 위험).

## 5.2 비선형 SVM 분류

선형적으로 분리되지 않는 데이터는 특성을 추가하여 고차원에서 분리할 수 있습니다.

### 5.2.1 커널 트릭 (Kernel Trick)
실제로 특성을 추가하지 않으면서도, 마치 수많은 고차원 특성을 추가한 것과 같은 효과를 내는 수학적 기법입니다.
* **다항식 커널 (Polynomial Kernel)**: `degree`를 통해 데이터셋을 복잡한 다항식 특성으로 변환한 효과를 냅니다.
* **가우스 RBF 커널 (Gaussian RBF Kernel)**: 샘플 간의 거리(유사도)를 기반으로 특성을 무한한 차원으로 확장하는 효과를 냅니다.
    * **하이퍼파라미터 $\gamma$ (감마)**: 샘플이 미치는 영향의 범위를 결정합니다. 값이 커지면 결정 경계가 개별 샘플에 민감해져 복잡해집니다.

## 5.3 SVM 회귀 (Regression)

분류와는 반대로, **제한된 마진 오류 안에서 도로 안에 가능한 한 많은 샘플이 들어가도록** 학습합니다.

* **$\epsilon$ (앱실론)**: 도로의 폭을 결정합니다. 마진(도로) 안에 샘플이 들어와도 손실(Error)이 발생하지 않는 **$\epsilon$-불감(epsilon-insensitive)** 특징을 가집니다.
* **SVR vs LinearSVR**:
    * `LinearSVR`: 훈련 세트 크기에 선형적으로 비례 (빠름).
    * `SVR`: 커널 트릭 지원, 데이터가 커지면 급격히 느려짐.

## 🛠️ 모델 선택 가이드 및 복잡도

| 모델 | 시간 복잡도 | 커널 지원 | 특징 |
| :--- | :--- | :--- | :--- |
| **LinearSVC** | $O(m \times n)$ | 아니오 | 매우 빠름, `liblinear` 기반 |
| **SVC** | $O(m^2 \times n)$ ~ $O(m^3 \times n)$ | **예** | 커널 트릭 사용 가능, `libsvm` 기반 |
| **LinearSVR** | $O(m \times n)$ | 아니오 | 선형 회귀에 최적화 |
| **SVR** | $O(m^2 \times n)$ ~ $O(m^3 \times n)$ | **예** | 비선형 회귀 지원 |

## 📝 목적 함수 (수학적 배경)

* **결정 함수**: $\hat{y} = \text{sign}(\mathbf{w}^T \mathbf{x} + b)$
* **소프트 마진 목적 함수**: 
  $$\text{minimize } \frac{1}{2} \mathbf{w}^T \mathbf{w} + C \sum_{i=1}^{m} \zeta_i$$
  * 조건: $y^{(i)}(\mathbf{w}^T \mathbf{x}^{(i)} + b) \geq 1 - \zeta_i$ 및 $\zeta_i \geq 0$
  * $\zeta_i$ (슬랙 변수): 각 샘플이 마진을 얼마나 위반할지 결정합니다.
