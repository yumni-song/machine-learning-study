# 📘 Machine Learning Study (Hands-On 기반 실습 정리)

## 📌 프로젝트 소개
이 저장소는 인공지능과 기계학습 수업 커리큘럼 및 실습 내용을 기반으로,  
이론과 코드 구현을 함께 학습하며 정리한 기록입니다.

## 🎯 학습 목표
- 머신러닝 기본 개념 이해
- 데이터 전처리 및 파이프라인 구축 능력 향상
- 다양한 모델 비교 및 성능 평가
- 실전 프로젝트 수행 능력 기르기
